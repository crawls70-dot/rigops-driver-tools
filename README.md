# RigOps — Truck Driver Command Center

RigOps is a free dashboard of tools for truck drivers and owner operators.

Tools included:
- Fuel Cost Calculator
- HOS Log Planner
- Pay Calculator
- Load Planner
- Trip Planner
- FMCSA Quick Reference

Goal: Help drivers calculate profits, plan trips, and operate more efficiently.

Built as a lightweight trucking operations dashboard.
